# Library Management System-Overview
A full-stack web application to manage library operations including book inventory, issuance, return tracking, and fine management with role-based access.

---

## Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MySql
- Authentication: JWT
- DevOps: Docker, Jenkins CI/CD

## Features
- JWT-based authentication and role based access control
- Book inventory management (add, update, categorize)
- Book issuance and return workflow
- Fine calculation for late returns
- Real-time stock tracking
- Admin dashboard with filtering and analytics
